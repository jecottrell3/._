INTRODUCTION

The Programs and Scripts in this Directory are intended as a Showcase for various Techniques and Technologies I have used over time. Please excuse any Bad Techniques; many of these were written long ago.

Programs are Rarely Finished, they are more often Abandoned. Comments Welcome.

CATALOG

atops
My version of a2ps, written to replace a crappy Printer Filter in SunOS and NeXT circa 1993. Notable for Postcript programming. Invoke by running "./atops < atops".

bmc
Not really all that special. Uses shell numeric variables.

fixown
Used for Renumbering UIDs on Disks. Should probably split out the Table into another file.

fixpath
Not much to see here either.

h2d and h2n
I put the MAC address (minus the colons) in /etc/hosts and generate the DHCP and DNS files from it. I use a special trick based on RFC 2137 to put all the PTRs in the Forward Domain. Invoke by "./h2n". The input is ./hosts; output is in "./homerj". Not sure why h2p is here.

kakuro
Builds a Table for Kakuro puzzles. Just run it to see what it does. Once I realized I could write this program this way, I just Owned It. Don't Try This At Home!

ks.cgi
Our Kickstart File CGI script. A Bit Old, but it shows how I structure and document code.

uxtract and murge
Not the Best Names. Used to Extract and Merge users, group, and shadow file entries. Documented in the murge program.

rr
Recursively Remove duplicate files in two trees. This program is so old it started as a csh script. Could stand a rewrite. Uses the questionable technique that next labels are Dynamically scoped.

tab
Print files using Aligned Tabs, much like how MySQL formats its query results. Try "./tab : < /etc/passwd".

up
Not sure what state this is in. Uses the idea between rsnapshot to build a deduped "hive" of backup files. The file lives in /back, so it's invoked as /back/up hosts....

zero-disks
I had to remove some (GPT?) Labels from a disk. This wipes the first and last GB of a disk.
